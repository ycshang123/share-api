# 内容分享应用后端微服务

## 1.user-service

#### 用户微服务：

主要提供

- 用户登录
- 查询/修改用户信息
- 用户收藏、投票积分记录查询

## 2.content-service

#### 内容微服务：

主要提供

- 用户登录
- 查询/修改用户信息
- 用户收藏、投票积分记录查询

## 3.notice-service

#### 公共微服务：

主要提供

- 管理员查询所有公告，显示/隐藏某个公告
- 新增、删除公告
- 查询最新公告
- 查询轮播公告

## 4.file-service

#### 文件微服务：

主要提供

- 单文件上传
- 文件批量上传

## 5.bonus-service

#### 积分微服务：

使用消息队列实现主要提供

- 积分增减服务

## 6.gateway-service

#### 网关微服务：

主要提供

- 路由转发
- 动态路由
- 统一认证鉴权
- 统一跨域处理
